# Symbl

An interpreted symbol-based language inspired by BF.

## Symbols

`>` increment pointer<br />
`<` decrement pointer<br />
`+` increment byte<br />
`-` decrement byte<br />
`*` lshift byte<br />
`/` rshift byte<br />
`.` output byte<br />
`:` output number<br />
`,` input byte<br />
`;` input number<br />
`[` start loop<br />
`]` end loop<br />
`a-zA-Z` set byte<br />
`0-9` set number<br />
`#` comment
